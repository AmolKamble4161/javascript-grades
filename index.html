Q] Write a code which can give greads to student according to their scores:

  • 90 - 100 : A
  • 70 - 89 : B
  • 60 - 79 : C
  • 50 - 59 : D
  • 00 - 49 : F

Ans :

The following code works step by step: first, it gets input from the user and stores that value in the marks variable. Then it checks the first condition to validate the input: marks > 100 OR marks < 0. If either is true, it logs the message “please enter valid input!” to the console.

If the value is between 0 and 100 (inclusive), it proceeds to the next conditions, which check the grades:

  • marks >= 90 && marks <= 100 → Grade A
  • marks >= 70 && marks <= 89 → Grade B
  • marks >= 60 && marks <= 69 → Grade C
  • marks >= 50 && marks <= 59 → Grade D
  • marks >= 0 && marks <= 49 → Grade F

This way, the program ensures that only valid input is accepted and then determines the correct grade based on the marks. 

let marks = prompt('Enter Marks!');

if(marks > 100 || marks < 0){
    console.log('please enter valid input!');
}else if(marks >= 90 && marks <=100){
    console.log('A');
}else if(marks >= 70 && marks <= 89){
    console.log('B');
}else if(marks >= 60 && marks <= 69){
    console.log('C');
}else if(marks >= 50 && marks <= 59){
    console.log('D');
}else if(marks >= 0 && marks <= 49){
    console.log('F');
}
